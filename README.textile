Blueprints is a collection of abstract data models, their implementations, their test suites, and supporting extensions. The following data models and implementations are provided in this package:

* Property Graph Model
** "TinkerGraph":http://wiki.github.com/tinkerpop/gremlin/tinkergraph in-memory graph
** "Neo4j":http://neo4j.org/ graph database
** "Sesame 2.0":http://www.openrdf.org compliant RDF stores
** "OrientDB":http://www.orientechnologies.com/ document-graph database
* Object Document Model
** TinkerDoc in-memory document store
** "MongoDB":http://mongodb.org document-oriented database
** "CouchDB":http://couchdb.apache.org/ document-oriented database (in design)
** "OrientDB":http://www.orientechnologies.com/ document-graph database (in design)

The following is a list of extensions for the respective abstract data models:

* Property Graph Model
** "Java Universal/Graph Framework":http://jung.sourceforge.net/ connector
